Drawing Sears Tower

Drawing the Sears Tower (known as Willis Tower these days) can be accomplished with four methods. The tower can be split into four distinct tiers. Each tier can then be drawn using a dedicated method. Further abstraction is possible but for now, let’s go with four distinct methods as follows:

  • masts() to draw the antennas on top of the tower;

  • top() to draw the top tier of the tower;

  • middle to draw the middle tier of the tower, and;

  • bottom() to draw the bottom tier.

The only variable that needs to be determined by the user, is the heigh of the tower, expressed in lines (remember, we are doing this as ASCII art.) The heigh is captured as static int N; in our program.

The rest of the dimensions, as shown in the schematic, are derived based on that variable N. The highlighted code below shows the corresponding calculations.

 1/**
 2 * Class with parameterized methods drawing the Sears tower.
 3 *
 4 * The class assembles a tower with three tiers: bottom, middle, and top. Each tier
 5 * is build in proportion to the total height of the tower -- specified in number
 6 * of lines to print. The proportions are captured in three groups of class variables.
 7 *
 8 * Class variables for height (suffixed _HEIGHT_FACTOR) specify the height of
 9 * each tier as a percentage of the tower's height, based on realistic measurements.
10 *
11 * Class variables for width (suffixed _WIDTH_FACTOR) specify the width of
12 * each tier as a percentage of the tower's height, based on a realistic aspect
13 * ratio of 1/6.45 (see https://skyscraperpage.com/cities/?buildingID=5).
14 *
15 * A third group of class variables is used for the symbols used in the drawing.
16 *
17 */
18public class SearsTower {
19
20    // Class variables for height
21    public static final double BOTTOM_HEIGHT_FACTOR = 0.45;
22    public static final double MIDDLE_HEIGHT_FACTOR = 0.36;
23    public static final double TOP_HEIGHT_FACTOR = 0.18;
24    public static final double WEST_ANTENNA_HEIGHT_FACTOR = 0.19;
25    public static final double EAST_ANTENNA_HEIGHT_FACTOR = 0.19;
26
27    // Class variables for width
28    public static final double BOTTOM_WIDTH_FACTOR = 1.0/6.45; // aspect ratio
29    public static final double MIDDLE_WIDTH_FACTOR = BOTTOM_WIDTH_FACTOR/2.0; // half aspect
30    public static final double TOP_WIDTH_FACTOR = MIDDLE_WIDTH_FACTOR/2.0; // quarter aspect
31
32    // Class variables for the symbols used in the drawing
33    public static final String BUILDING_ELEMENT = "H";
34    public static final String ANTENNA_ELEMENT = "|";
35
36    /**
37     * Prints the bottom tier of the tower, in proportion to a
38     * user specified height.
39     * @param towerHeight the height of the tower in number of lines to print
40     */
41    public static void bottomTier(int towerHeight) {
42        // How tall is this section in proportion to the tower's height?
43        int bottomTierHeight = (int) (BOTTOM_HEIGHT_FACTOR * towerHeight);
44        // How wide is this section in proportion to the tower's height?
45        int bottomTierWidth = (int) (BOTTOM_WIDTH_FACTOR * towerHeight);
46        for (int i = 0; i < bottomTierHeight; i++) {
47            System.out.println(BUILDING_ELEMENT.repeat(bottomTierWidth));
48        }
49    } // method bottomTier
50
51    public static void middleTier(int towerHeight) {
52        int middleTierHeight = (int) (MIDDLE_HEIGHT_FACTOR * towerHeight);
53        int middleTierWidth = (int) (MIDDLE_WIDTH_FACTOR * towerHeight);
54        for (int i = 0; i < middleTierHeight; i++) {
55            System.out.println(BUILDING_ELEMENT.repeat(middleTierWidth));
56        }
57    } // method middleTier
58
59    public static void topTier(int towerHeight) {
60        int topTierHeight = (int) (TOP_HEIGHT_FACTOR * towerHeight);
61        int topTierWidth = (int) (TOP_WIDTH_FACTOR * towerHeight);
62        for (int i = 0; i < topTierHeight; i++) {
63            System.out.println(BUILDING_ELEMENT.repeat(topTierWidth));
64        }
65    } // method topTier
66
67    // driver method (main)
68    public static void main(String[] args) {
69        int N = 40; // desired height
70        topTier(N); // top tier for the desired height
71        middleTier(N); // middle tier for the desired height
72        bottomTier(N); // bottom tier for the desired height
73    }
74}
