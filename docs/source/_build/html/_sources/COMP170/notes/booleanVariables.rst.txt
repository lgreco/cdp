Boolean Variables
=====================

Boolean variables are one of the most useful primitive data types in programming. And in the beginning they can be a bit difficult to comprehend. That's because for many programmers the terms boolean *variables* and boolean *expressions* mean the same thing. And when some of these programmers walk into a classroom to teach programming, they (we) forget to make the distinction. So, let's start with a boolean variable:

.. code-block:: java

   boolean isChicagoAwesome = true;
